Key Capabilities

Everything you need to build, test, and run a live trading system on Upstox.

📡

Real-Time Streaming

Protobuf-decoded WebSocket ticks via MarketDataStreamerV3. Supports ltpc, full, and option_greeks modes with smart per-instrument routing.

🧠

Pluggable Strategies

Extend BaseStrategy with three methods. SuperTrend Pro v6.3 and ScalpPro v1.0 are built-in. Register your own in one line.

🛡️

Risk Management

Daily loss caps, per-trade risk limits, concurrent position guards, trading side restrictions (LONG_ONLY / SHORT_ONLY / BOTH), and forced square-off.

📊

Option Chain Analytics

PCR, Max Pain, OI concentration, IV skew, and live Greeks overlay — computed in pure Python with no API side effects.

📄

Paper Trading Default

Ships with PAPER_TRADING=True. Simulate every strategy safely before switching to live with a single config toggle.

⏱️

IST-Aligned Scheduler

APScheduler with Asia/Kolkata timezone. Candle checks every minute during market hours (9:15–15:30), auto square-off at 15:15.

🎛️

Web Dashboard

Interactive TradingView Lightweight Charts, live trade log, strategy parameter editor, and real-time WebSocket tick updates.

🌐

Proxy Support

Full proxy layer for static IP whitelisting compliance. Supports both SDK-level and process-level proxy injection with REQUIRE_UPSTOX_PROXY strict mode.

🔔

Multi-Channel Alerts

Email (SMTP) notifications out of the box. Telegram and other providers are supported via the pluggable NotificationProvider interface.

Built-in Strategies

Two production-ready strategies included. Plug in your own with three methods.

Primary

SuperTrend Pro v6.3

Faithful 1:1 Pine Script port (590 lines)

  • H1: Dual SuperTrend agreement gate
  • H2: Consecutive bar confirmation gate
  • H3: Higher-timeframe trend filter
  • Soft Score: ADX + volume + ATR + ROC + BB squeeze
  • Exit A/B/C: ATR SL+TP, ATR SL + Profit%, or Trailing
  • Auto TF: Thresholds adapt to selected timeframe
Scalping

ScalpPro v1.0

High-speed 1-minute EMA + VWAP + RSI

  • Entry: EMA(9)/EMA(21) crossover signal
  • Filter: Anchored VWAP trend + RSI confirmation
  • Gate: ADX strength threshold
  • Exits: 3-tier partial booking (40% / 40% / 20%) + breakeven SL
  • Swarm: 1–5 parallel lots per signal
  • Options: Liquidity filter, spread limits, adaptive relaxation

Quick Start

Up and running in under 5 minutes.

1

Clone & install

git clone https://github.com/sachdev27/Upstox-Trigger.git
cd Upstox-Trigger
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
2

Configure

cp .env.example .env
# Add your Upstox API_KEY, API_SECRET, REDIRECT_URI
3

Run

bash start_terminal.sh
# Server on http://localhost:8210
4

Authenticate & trade

# Login via OAuth
open http://localhost:8210/auth/login

# Open dashboard
open http://localhost:8210/dashboard
